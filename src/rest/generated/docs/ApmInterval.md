
# ApmInterval

A `[startTime, endTime)` half-open window. Used both as the per-record time slice on `MetricsRecord` and as the response-level executed-window echo. RFC3339, startTime inclusive, endTime exclusive. 

## Properties

Name | Type
------------ | -------------
`startTime` | string
`endTime` | string
